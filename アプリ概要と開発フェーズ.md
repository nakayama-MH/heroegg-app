# アプリ概要と開発フェーズ

本アプリは、認証後にホームから各機能へアクセスし、イベント情報閲覧、プロフィール管理、施設の入退館、問い合わせ、設定を提供する。位置情報を用いて近隣のEggを表示する。

# 画面遷移フロー

- アカウント有無判定
    - yes → ログイン画面
    - no → 会員登録画面
- 会員登録画面 → ログイン画面（接続あり）
- ログイン画面 → パスワード再発行（接続あり）
- ログイン → アカウント認証と種別判定
    - no → ログイン画面に戻る
    - yes → ホーム画面
- ホーム画面 → 以下の主要機能
    - プロフィール
    - peetixイベント取得表示
    - 入館&#x2F;チェックイン
    - お問い合わせ
    - 各種設定

# 各画面の機能詳細

## 認証関連画面

- ログイン: メール&#x2F;パスワード入力、送信、再発行リンク
- 会員登録: 必須項目入力、規約同意、登録完了後ログイン導線
- パスワード再発行: メール送信、リセット完了通知

## ホーム画面

- 位置情報取得の許諾&#x2F;再設定
- 近隣のEgg一覧表示

## プロフィール画面

- 会員ランク（バッジ表記）
- 参加履歴&#x2F;入館履歴
- プロフィール情報閲覧・編集
- my施設の表示

## peetixイベント機能

- イベント情報一覧表示（開催日・場所・状態）
- 詳細外部サイトへ遷移

## 施設入退館機能

- 入館&#x2F;チェックイン: 施設入館時にチェックインアクション
- 退館&#x2F;チェックアウト: 施設退館時にチェックアウトアクション

## お問い合わせ機能

- 各種お問い合わせフォーム
- 共創パートナー募集フォーム

## 各種設定

- アカウント情報、通知、位置情報権限などの設定

# 開発スケジュール

- 第一フェーズ（〜4&#x2F;1）
    - 認証一式（判定&#x2F;登録&#x2F;ログイン&#x2F;再発行&#x2F;種別判定）
    - ホーム（位置情報・近隣Egg表示）
    - peetixイベント一覧と外部遷移
    - 基本プロフィール表示、各種設定、お問い合わせ
- 第二フェーズ（〜8&#x2F;31）
    - 施設入退館機能（ダウンロード条件可否により実装）
    - 入退館履歴のプロフィール連携、バッジ&#x2F;ランク表示強化、my施設
- 第三フェーズ（年内予定）
    - 安定化・運用改善・分析基盤連携などの拡張（範囲は第二フェーズの結果に基づき確定）
